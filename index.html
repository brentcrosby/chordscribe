How to use ChordScribe

Editor view

  • Type lyrics like a normal text editor — Enter starts a new line, and you can select, cut, copy & paste across lines.
  • Hover a word and click the + above it to add a chord. Click an existing chord to change it, or drag it to move it.
  • Type a name like Chorus or Verse 2 on its own line to turn it into a section heading.

ChordPro view

  • Edit the raw source directly. Put chords inline in brackets like [D] before the word they land on.
  • Mark sections with {comment: Verse 1}. Changes sync back to the Editor view.

Song details

  • Fill in the title, artist, key and other fields at the top. Add a Road map (e.g. Intro V1 CH V2) and Notes for quick reminders.

Keyboard shortcuts

  • Ctrl/Cmd + Z — Undo
  • Ctrl/Cmd + Shift + Z (or Ctrl/Cmd + Y) — Redo
  • Esc — close a chord editor or this dialog

Saving & sharing

  • Use Export to print, save a PDF, or download the ChordPro source. Import loads .txt, .pro, .chordpro or .crd files — and can read back a .pdf that ChordScribe printed (chords sit above the words, so the layout is recovered from the page).
